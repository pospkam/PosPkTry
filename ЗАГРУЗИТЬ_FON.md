# Загрузите фоновое изображение FON

## Файл не найден в проекте

Пожалуйста, загрузите изображение FON в проект:

### Куда загрузить:
```
/workspace/public/fon.jpg
```
или
```
/workspace/public/fon.png
```

### Откуда взять изображение:

1. **Unsplash (бесплатно)**:
   - Камчатка: https://unsplash.com/s/photos/kamchatka
   - Вулканы: https://unsplash.com/s/photos/volcano
   - Горы: https://unsplash.com/s/photos/mountains-sunrise

2. **Pexels (бесплатно)**:
   - https://www.pexels.com/search/kamchatka/
   - https://www.pexels.com/search/volcano%20sunrise/

3. **Рекомендованные изображения**:
   - Восход солнца над вулканами Камчатки
   - Горизонтальное изображение
   - Размер: минимум 1920x1080px
   - Формат: JPG или PNG

### После загрузки:

Файл будет автоматически использоваться как фон на всю ширину.

### Команда для загрузки (если у вас есть ссылка):

```bash
cd /workspace/public
wget -O fon.jpg "ССЫЛКА_НА_ИЗОБРАЖЕНИЕ"
```
