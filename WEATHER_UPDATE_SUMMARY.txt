⚙️ БЫСТРАЯ НАСТРОЙКА:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Добавьте ключи в .env (опционально):
     
     OPENWEATHERMAP_API_KEY=your_key
     WEATHERAPI_KEY=your_key
     YANDEX_WEATHER_API_KEY=your_key

  2. Используйте виджет:
     
     import { WeatherWidget } from '@/components/WeatherWidget';
     
     <WeatherWidget
       lat={53.0}
       lng={158.6}
       location="Петропавловск-Камчатский"
     />

  3. Или через API:
     
     GET /api/weather?lat=53.0&lng=158.6
