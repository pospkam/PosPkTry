## 3. ПОГОДНЫЕ АНИМАЦИИ

### Дождь
```css
.rain-drop {
  animation: rain-fall 0.5s linear infinite;
}
```
- Капли падают сверху вниз
- Прозрачность 0.6
- Случайные позиции

### Снег
```css
.snowflake {
  animation: snow-fall 3s linear infinite;
}
```
- Снежинки медленно падают
- Покачивание (swing)
- Разные размеры

### Ветер
```css
.wind-line {
  animation: wind-blow 1s ease-in-out infinite;
}
```
- Горизонтальные линии
- Скорость зависит от силы ветра
