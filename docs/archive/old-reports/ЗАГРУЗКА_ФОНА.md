## Рекомендуемые параметры:

- **Формат**: JPG или PNG
- **Размер**: минимум 1920x1080 (Full HD)
- **Оптимально**: 2560x1440 или 3840x2160
- **Вес файла**: до 2MB (сжатое)

## Тематика для Камчатки:

1. Вулканы (Авачинский, Ключевская сопка)
2. Гейзеры (Долина Гейзеров)
3. Океан и бухты
4. Медведи
5. Северное сияние
6. Снежные пейзажи

## После загрузки:

1. Файл будет доступен в `/public/uploads/`
2. Нужно будет обновить компонент `SamsungWeatherDynamic.tsx`
3. Добавить путь к изображению в CSS
